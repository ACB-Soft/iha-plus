# İHA Plus

## 📦 Kurulum

Projeyi yerel ortamınızda çalıştırmak için:

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

3. Üretim için derleme yapın:
   ```bash
   npm run build
   ```

## 📄 Lisans

Bu proje **ACB_Soft** tarafından geliştirilmiştir. Tüm hakları saklıdır.
